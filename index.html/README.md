# 🧠 Conceptos Matemáticos Aplicados

## 🔢 Fibonacci

La secuencia de Fibonacci sigue la fórmula:

F(n) = F(n-1) + F(n-2)

Ejemplo:

1 → 1 → 2 → 3 → 5 → 8 → 13 → 21

Aplicado en:

* Finanzas
* Crecimiento progresivo
* Algoritmos adaptativos

---

## 🛡️ Números Primos

Los números primos solo son divisibles entre:

* 1
* Ellos mismos

Ejemplos:

* 7
* 97
* 739
* 8293

Aplicado en:

* Seguridad digital
* Criptografía RSA
* Generación de claves
