Introducción

Este proyecto consiste en el desarrollo de una aplicación web interactiva orientada a la demostración práctica de conceptos matemáticos y computacionales relacionados con:

La sucesión de Fibonacci
Los números primos
La criptografía moderna
La seguridad informática
Simulación de algoritmos matemáticos

La plataforma integra elementos visuales, simulaciones dinámicas y algoritmos implementados en JavaScript con el objetivo de explicar de manera didáctica cómo estas estructuras matemáticas poseen aplicaciones reales en áreas como las finanzas, la ingeniería y la protección de datos.

Objetivos del Proyecto
Objetivo General

Desarrollar una página web interactiva que permita visualizar aplicaciones prácticas de la sucesión de Fibonacci y los números primos en contextos de ahorro financiero y criptografía computacional.

Objetivos Específicos
Implementar algoritmos matemáticos utilizando JavaScript.
Simular un sistema de ahorro progresivo basado en Fibonacci.
Verificar números primos mediante algoritmos eficientes.
Generar claves pseudoaleatorias utilizando combinaciones matemáticas.
Diseñar una interfaz moderna e interactiva orientada a la experiencia del usuario.

Descripción de los Módulos
1. Módulo de Fibonacci

Este apartado presenta una simulación de ahorro progresivo utilizando la sucesión de Fibonacci.

La lógica del sistema consiste en calcular cada nuevo depósito como la suma de los dos anteriores, generando así un crecimiento gradual y acumulativo.

Aplicaciones Representadas
Finanzas personales
Modelos de crecimiento
Algoritmos recursivos
Optimización matemática
Algoritmo Utilizado

El sistema implementa un algoritmo iterativo que calcula dinámicamente cada término de la secuencia.

Ejemplo:

1, 1, 2, 3, 5, 8, 13, 21...
2. Módulo de Verificación de Números Primos

Este módulo permite validar si un número ingresado por el usuario es primo.

Se utiliza un algoritmo optimizado basado en el método:

6k ± 1

el cual reduce significativamente la cantidad de verificaciones necesarias.

Aplicaciones Representadas
Criptografía RSA
Seguridad informática
Sistemas de autenticación
Generación de claves
Funcionalidades
Detección de divisores
Validación matemática
Evaluación de seguridad
Resultados dinámicos en pantalla
3. Módulo de Criptografía Simbiótica

Esta sección combina la secuencia de Fibonacci y los números primos para simular la generación de tokens criptográficos.

El sistema:

Genera iteraciones Fibonacci aleatorias.
Selecciona números primos de una base de datos interna.
Calcula un producto criptográfico.
Produce una llave pseudoaleatoria en formato estructurado.
Objetivo del Módulo

Demostrar de manera conceptual cómo distintas estructuras matemáticas pueden combinarse para fortalecer mecanismos de seguridad digital.
